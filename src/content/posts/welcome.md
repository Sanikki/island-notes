# 欢迎来到小岛笔记 🏝️

你好呀，远道而来的旅人！👋

这里是我的个人博客，一个**动物森友会风格**的小岛，用来记录我在前端与技术世界里的「钓鱼」与「挖化石」——也就是平时的学习笔记和踩坑记录。

> 🌿 在小岛上，最好的节奏不是最快，而是刚刚好。

## 🗺️ 这个小岛能做什么

- **文章列表**：首页按时间倒序排列所有笔记
- **文章详情**：点击任意卡片即可阅读完整 Markdown 内容
- **标签分类**：按技术主题检索，同一个坑不必踩两次
- **关于页面**：了解这座岛，以及SANIKKI是谁

## 📝 为什么用 Markdown

因为写作应当像写便签一样轻松：

```md
# 今天学到了什么

- 用 `import.meta.glob` 批量导入文章
- 用 `react-markdown` 渲染内容
```

| 工具 | 用途 |
| --- | --- |
| Vite | 极速开发服务器 |
| React | 组件化 UI |
| animal-island-ui | 动森风组件库 |
| react-markdown | 渲染 Markdown |
